# Code signing policy

Free code signing provided by [SignPath.io](https://about.signpath.io/), certificate by [SignPath Foundation](https://signpath.org/terms.html).

## Roles

- Committer and reviewer: [d0d1](https://github.com/d0d1)
- Approver for signing requests: [d0d1](https://github.com/d0d1)

## Privacy policy

This program will not transfer any information to other networked systems unless specifically requested by the user or the person installing or operating it.

## Release policy

- Only artifacts built from this repository's source code and release workflow may be submitted for signing.
- Signed release assets are published through the GitHub release workflow after the signing request completes.
- Each signing request requires manual approval before release publication.
